cortex.io._download.download(source: str | pathlib.Path, destination: str | pathlib.Path, filename: str | None = None, checksum: str | None = None, maximum_redirect_url_hops: int = 3, boto3_s3_client: botocore.client.BaseClient | None = None)¶

Download a file from a URL and save it to a local path. Supports standard URLs, Google Drive, and S3.

Parameters:

  • source (str | Path) – The URL of the file to be downloaded.

  • destination (str | Path) – The local directory where the downloaded file will be saved.

  • filename (str | None, optional) – The name of the file to be saved, by default None.

  • checksum (str | None, optional) – The checksum of the file to be downloaded, by default None.

  • maximum_redirect_url_hops (int, optional) – The maximum number of hops to follow when resolving a URL redirect, by default 3.

  • boto3_s3_client (BaseClient | None, optional) – The boto S3 client to use, by default None.

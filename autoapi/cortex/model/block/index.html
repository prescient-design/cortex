Bases: torch.nn.Module

1D explicit convolution pre-norm residual block with optional layer-norm and swish activation. Each block has two convolution layers. The order of operations is norm -> act -> conv -> norm -> act -> conv -> add.

This module expects inputs to be a tuple of (features, padding_mask).

Dropout is applied to the output of the block.
